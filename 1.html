태그 이름은 그 속성과 비슷한 유사한/비유적 이름 사용하여 만들어 사람들이 알기 쉽게 만들었다.
처음에 배우는 것은 배우기 쉽지만, 가장 많이 사용하고 중요하다.

그 누구도 정보로부터 소외받지 않도록 노력해야 한다.

태그

  • 글씨를 진하게 표시 방법: < strong >대상< /strong >
  • 밑줄(underline) 표시 방법: < u >대상< /u >
  • 제목 표시 방법(개행, 숫자가 높을수록 굵기가 얇다 h1~h6): < h1 >대상< /h1 >
  • 줄바꿈 표시 방법: 대상< br >대상
  • 단락 표시 방법: < p >대상< /p >
  • 이미지 표시 방법(속성:src img태그에 파일 디렉토리 경로에있는 어떠한 이미지를 지성하여 웹 페이지에 결과물을 출력할때 사용됩니다, width 폭, height 높이,): < img src="파일명" width="% or px;" height="% or px;" >
  • 목차 표시 방법(dl:Description List 개념과 정의, ul:Unorder List 순서가 없는, ol:order List 순서가 있는):
    순서 있는 경우, 없는 경우
    < ul or ol >
     < li >목차 이름< /li >
    < /ur or ol >
    개념과 정의 (dt와 dd는 1:1 뿐 아니라 다:다의 형식도 가질 수 있음)
    < dl >
     < dt >인라인 요소(개념)< /dl >
      < dd >블럭 요소(정의)< /dd >
    < /dl >
  • 스페이스 표시 방법(공백의차이 & nbsp;<& ensp;<& emsp: ): & nbsp;대상
  • 표 표시 방법(table, tr, td):
     < table >
      < tr >
       < td >대상< /td >
       < td >대상의 수치< /td >
      < /tr >
     < /table >
  • 링크 걸기(속성:target="_blank"(새탭), title+"링크사이트"(툴팁)): < a href="사이트 주소" > 링크 걸기 대상 < a >
